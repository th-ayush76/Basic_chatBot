# Basic Chatbot 🤖

A simple and interactive command-line chatbot built with Python. This project demonstrates basic natural language processing logic and rule-based responses.

## 🚀 Features
* **Conversational Interface:** Responds to user inputs in real-time.
* **Predefined Logic:** Handles basic greetings, common questions, and closing statements.
* **Lightweight:** Runs directly in your terminal with no external heavy dependencies.

## 🛠️ How to Run
1. Ensure you have [Python](https://www.python.org/) installed on your system.
2. Clone this repository or download the source code.
3. Open your terminal or command prompt in the project folder.
4. Run the chatbot using the following command:
   ```bash
   python basicchatbot.py
